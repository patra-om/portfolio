# omptra.github.io

My personal portfolio as a 2nd year Robotics & AI student. This site combines modern web aesthetics with a high-tech "cybernetic" theme.

🔗 **Live:** [omptra.github.io](https://omptra.github.io)

## Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Animations:** [GSAP](https://greensock.com/gsap/) (ScrollTrigger, ScrollTo, Custom Sequences)
- **Interactions:** [Vanilla Tilt](https://micku7zu.github.io/vanilla-tilt.js/)
- **Visuals:** Font Awesome, Google Fonts (Fraunces & Inter)

## Features

- 🛡️ **Cybernetic Scanner:** A high-tech HUD-style landing animation on boot.
- 🌫️ **Glassmorphism:** Modern UI with blurred backgrounds and animated glows.
- ⌨️ **Terminal Effects:** Typewriter animations on section headings.
- 🖱️ **Custom Cursor:** Interactive dual-layered cursor for desktop.
- 📱 **Fully Responsive:** Optimized for all screen sizes (mobile, tablet, desktop).
- 🔄 **GSAP Reveal:** Smooth scroll-triggered section transitions.
